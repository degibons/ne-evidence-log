# ne-evidence-log

Публичный журнал хешей доказательственных snapshot.

Этот репозиторий предназначен для публикации:

- идентификаторов snapshot;
- дат;
- SHA-256 хешей архивов из приватного репозитория;
- SHA-256 хешей файла `SHA256SUMS.txt`;
- ссылок на commit приватного репозитория.

Публичный журнал предназначен для фиксации момента существования соответствующего приватного архива без публикации его содержимого.
